# Changelog

All notable changes to AxionAx SDK TypeScript will be documented in this file.

## [1.8.0] - 2025-11-15

### Added
- Cross-platform installation scripts
- TypeScript examples (send transaction, smart contract, event listener)
- ESLint and Prettier configurations
- EditorConfig for consistent style
- MIT License
- Optimized TypeScript configuration (ES2021, ESNext)

### Changed
- Enhanced TypeScript compiler options for better type safety
- Enabled source maps and declaration maps

## [1.0.0] - 2025-10-01

### Added
- Initial release
- AxionAxProvider for blockchain interaction
- Wallet management
- Transaction signing and sending
- Smart contract interaction support
- Event subscription
- TypeScript definitions
